Definición de CSS

CSS, o Hojas de Estilo en Cascada, es un lenguaje utilizado para describir la presentación de un documento escrito en HTML o XML. CSS define cómo se deben mostrar los elementos en la pantalla, en papel o en otros medios.

Funcionamiento Básico de CSS

CSS funciona aplicando estilos a los elementos HTML de una página. Los estilos se pueden definir de manera global o específica para ciertos elementos, permitiendo así una gran flexibilidad en el diseño de las páginas web.

Incluir CSS en un archivo HTML

Para incluir CSS en un archivo HTML, se pueden usar tres métodos: incrustar el CSS directamente en el HTML utilizando la etiqueta `style`, enlazar un archivo CSS externo con la etiqueta `link`, o aplicar estilos directamente a elementos mediante el atributo `style`.

Definición de una regla CSS

Una regla CSS se compone de un selector y un bloque de declaración. El selector define qué elementos HTML serán afectados, mientras que el bloque de declaración contiene las propiedades y valores que se aplicarán a esos elementos.

Elemento de una regla CSS

Una regla CSS está compuesta por los siguientes elementos:

  • Regla: Conjunto completo que incluye el selector y el bloque de declaración.
  • Selector: Parte de la regla que define a qué elementos HTML se aplicarán los estilos.
  • Bloque de declaración: Contiene uno o más pares de propiedad-valor, encerrados entre llaves `{}`.
  • Propiedad: Aspecto del elemento que se desea estilizar, como `color`, `font-size`, etc.
  • Valor: Especifica el valor que se aplicará a la propiedad, como `red`, `16px`, etc.

Medios CSS

Los medios CSS permiten aplicar estilos específicos según el tipo de dispositivo o las condiciones de visualización, como la resolución de pantalla o el tamaño de la ventana. Se utilizan en las consultas de medios (media queries) para crear diseños responsivos.

Comentarios CSS

Los comentarios en CSS se utilizan para agregar notas dentro del código que no se muestran en la página web. Se escriben entre `/*` y `*/`. Los comentarios son útiles para explicar partes del código o para desactivar temporalmente reglas.

Sintaxis de la definición de las propiedades CSS

La sintaxis para definir las propiedades CSS dentro de un bloque de declaración es la siguiente: code propiedad: valor;/code. Cada propiedad se define con su nombre seguido de dos puntos y el valor que se desea aplicar, finalizando con un punto y coma.
